# 🚀 Dispo+

> A modern disposition assistant designed to streamline compliance screening workflows with intelligent templates, automation, and productivity tools.

## 📖 Overview

Dispo+ is a web-based productivity tool built to help compliance analysts perform faster and more consistent disposition writing during screening and due diligence processes.

The application provides reusable templates, automated calculations, and workflow enhancements that reduce manual effort while improving consistency and accuracy.

---

## ✨ Features

- ✅ Modern responsive interface
- 🔐 User authentication with Supabase
- 👤 Secure login and logout
- 💾 Persistent user sessions
- 🌙 Dark mode support
- 📋 One-click copy to clipboard
- 💾 Automatic local storage saving
- ⚡ Fast template generation
- 🌍 Country-specific disposition support
- 📊 Common Name Frequency Checker
- 👨 Father's Name Mismatch Generator
- ⚖️ Maximum Penalty Calculator
- 📝 Potential Match Template Generator
- ⏰ Live Time Display
- 🎨 Premium UI Design
